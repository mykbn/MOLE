Clock Faces

FlipClock.js has a fully extendible API to create any clock face you need. More clock faces will be added over time, so be sure to fork the code and make a pull request if you want to see your clock face appear in the core library.

Creating Your Clock Face

Be sure to name your clock face correctly. Your clock face name should be in the following format:
FlipClock.YourClockFace - and your filename would be yourclock.js (if create one).


FlipClock.YourClockFace = FlipClock.Face.extend({
	
	build: function() {
		this.log('This method is used to build your clock face.');
	},
	
	reset: function() {
		this.log('This method is triggered when the clock is reset');
	},
	
	start: function() {
		this.log('This method is triggered when the clock is started');
	},
	
	stop: function() {
		this.log('This method is triggered when the clock is stopped');
	},
	
	increment: function() {
		this.log('This method is used to increment your clock face.');
	},

	flip: function() {
		this.log('This method is triggered when the clock is flipped');
	}
	
});

Refer to the Developer API Referrence for more information regarding object properties and methods.
